### Map-Fly Bee

### 起步

```
npm install 
```
```
npm start
```
#### 功能

* 距离测量
  * AMap.RangingTool
* 区域划分
  * AMap.MouseTool
* 点的查询
  * AMap.Marker + AMap.GeometryUtil.isPointInRing方法判断点是否在面内
* 区域边界拖动编辑
  * AMap.PolyEditor
